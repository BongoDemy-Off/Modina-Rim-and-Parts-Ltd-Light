# Modina Rim & Parts Ltd.

![Modina Banner](https://images.unsplash.com/photo-1558618666-fcd25c85cd64?w=1200&q=80)

> Official corporate website for Modina Rim & Parts Ltd., an ISO-certified manufacturer of premium automotive components in Bangladesh.

## Overview
Modina Rim & Parts Ltd. is an industry-leading manufacturer of premium motorcycle rims, rickshaw rims, and automotive hardware. This repository contains the source code for the corporate website, showcasing our product catalog, manufacturing capabilities, and quality certifications.

## Tech Stack
- **Frontend Framework:** React 19 + Vite
- **Styling:** Tailwind CSS v4
- **Routing:** React Router v7
- **Animations:** Motion (Framer Motion)
- **Icons:** Lucide React
- **Language:** TypeScript

## Quick Start

### Prerequisites
- Node.js 18+ 
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/BongoDemy-Off/modina-rim-parts-ltd.git
   cd modina-rim-parts-ltd
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```

5. Preview production build:
   ```bash
   npm run preview
   ```

## Project Structure
- `/src/components`: Reusable UI components
- `/src/pages`: Page-level components
- `/src/assets`: Static assets like images and fonts
- `/public`: Public assets
